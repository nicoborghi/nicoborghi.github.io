# Nicola Borghi - Personal Website

Personal website built with HTML5 UP's Dimension template.

- [https://nicoborghi.github.io](https://nicoborghi.github.io)

## Local Development
Open `index.html` in a web browser. For SASS changes, compile `assets/sass/main.scss` to `assets/css/main.css`.
